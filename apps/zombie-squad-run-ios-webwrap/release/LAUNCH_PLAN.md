# Paid vs Free Launch Plan

## Option A: Paid launch
### Best for
- fastest release path
- least SDK complexity
- validating willingness to pay

### Suggested pricing
- $0.99 or $1.99 one-time

### Pros
- simpler launch
- no ads integration required
- no IAP required on day one

### Cons
- fewer downloads
- harder to grow organically

## Option B: Free launch
### Best for
- maximizing downloads
- testing retention first

### Suggested monetization
- rewarded revive later
- cosmetic squad packs
- no-ads unlock

### Pros
- easier top-of-funnel
- better for later optimization

### Cons
- more implementation work
- ad / IAP setup needed

## Recommendation
For the absolute fastest launch from Windows-only tooling, start with a **paid launch**.
After the release flow is stable, add rewarded revive and cosmetic upsells in a later update.
