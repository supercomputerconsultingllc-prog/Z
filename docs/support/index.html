Zombie Squad Run

Support

Zombie Squad Run support covers gameplay help, bug reports, account-less local progress questions, and business inquiries.

Contact

  • Use the published support page linked from the App Store listing for release support.
  • If a direct contact email is added before launch, update this page and the hosted support page in the iOS web-wrap release bundle.

What to include in a support request

  • device model
  • iOS version
  • app version or build number
  • short description of the problem
  • screenshot or screen recording if available

Common issues

Lost progress

The current version stores best score and coin balance locally on the device. Removing browser data or reinstalling the wrapped app may clear local progress.

Gameplay feedback

If a wall feels unfair, revive timing feels too punishing, or one lane is misbehaving, include the wave number and what happened right before the issue.

Business

For licensing, publishing, or collaboration inquiries, use the support email with a clear subject line.
