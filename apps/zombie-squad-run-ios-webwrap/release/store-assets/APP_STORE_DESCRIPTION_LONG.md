# Long Description Draft

Zombie Squad Run is a fast lane-based action game where every shot matters.

Run forward with your squad, choose the lane to fire into, and survive a chaotic zombie run filled with walls, upgrades, revives, and coin-driven progression. Shoot zombie packs to grow your team, break walls before impact, collect gun crates to increase firepower, and push for a higher distance every run.

## What makes it fun
- simple controls with quick decision-making
- squad growth through successful plays
- wall-breaking risk that creates tension
- upgrade loop with coins, revives, and firepower
- short runs that are easy to replay

## Core features
- three-lane runner shooter gameplay
- zombies, walls, gun crates, and coin pickups
- revive mechanic
- mission-based retention hooks
- persistent best score and coin balance

Build your squad, shoot the right lane, and see how far you can survive.
