# Zombie Squad Run iOS Web Wrap Quickstart

## Windows fastest path
1. Install Node.js
2. Open this folder in a terminal
3. Run `npm install`
4. Run `npm run release:prep`
5. Push this folder to GitHub
6. Connect the repo to Codemagic
7. Configure Apple credentials in Codemagic
8. Build and export the iOS app in the cloud

## Most useful commands
```bash
npm install
npm run prepare:web
npm run release:prep
npm run preflight
npm run cap:init:notes
```
