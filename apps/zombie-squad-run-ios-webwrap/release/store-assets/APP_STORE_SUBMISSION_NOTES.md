# App Store Submission Notes

## Required fields to finalize
- app name
- subtitle
- support URL
- privacy policy URL
- age rating answers
- pricing
- territories

## Suggested first-release pricing
- paid launch at $0.99 or $1.99 for fastest path

## Review-sensitive notes
- if using ads later, update privacy and disclosures
- if using rewarded revive later, disclose relevant tracking or ad SDK behavior
- if targeting younger players, avoid misleading monetization patterns

## Before submission
- confirm screenshots match actual current UI
- confirm support and privacy URLs are live
- proofread listing copy one last time
