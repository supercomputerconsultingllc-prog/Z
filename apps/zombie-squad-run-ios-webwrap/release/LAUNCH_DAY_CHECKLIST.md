# Launch Day Checklist

## Before submit
- [ ] screenshots uploaded
- [ ] support URL live
- [ ] privacy policy URL live
- [ ] app icon uploaded
- [ ] pricing chosen
- [ ] age rating reviewed
- [ ] metadata proofread

## Build and submit
- [ ] final cloud build succeeds
- [ ] signed `.ipa` archived
- [ ] upload to App Store Connect succeeds
- [ ] build selected in app listing
- [ ] submit for review

## After submit
- [ ] save release notes
- [ ] monitor review status
- [ ] prepare first update list
- [ ] prepare support responses for common questions

## After approval
- [ ] verify listing is live
- [ ] test install from App Store
- [ ] confirm pricing and region settings
- [ ] collect first player feedback
