# Zombie Squad Run Monetization Plan

## Best launch order
### Version 1
- paid app launch at $0.99 or $1.99
- no ads SDK yet
- focus on shipping fast and validating installs

### Version 1.1
- rewarded revive
- optional ad-free / premium support unlock if launch switches to free later

### Version 1.2
- cosmetic squad skins
- cosmetic weapon skins
- seasonal mission pack

## Why this order works
- simplest release path first
- less SDK and policy complexity
- faster iteration after the release process is proven

## Core monetization hooks already aligned with gameplay
- revive moment after a failed run
- upgrade loop and coin economy
- squad skin or weapon skin cosmetics
- premium stage themes later
