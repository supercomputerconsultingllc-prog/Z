# Privacy Policy Draft

Zombie Squad Run does not require account creation to play the current local browser-based build.

## Information stored on device
- best score
- coin balance
- sound preference
- mission progress

This information is stored locally on the player's device using browser storage so the game can remember progress and settings between sessions.

## Information not currently collected
The current build does not require sign-in and does not intentionally send gameplay data, personal information, or device analytics to a remote server.

## Children's privacy
This draft does not describe child-directed data handling. Final age-rating and privacy disclosures should match the exact release configuration.

## Support and privacy contact
For privacy questions or data-handling questions related to the release build, direct players to the published support contact listed on the app's support page.

## Future release note
If analytics, ads, crash reporting, cloud saves, account systems, or in-app purchases are added, this policy must be updated before release and the App Store privacy answers must be reviewed again.
