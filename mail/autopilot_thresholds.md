# Autopilot Threshold Rules

Auto-reply when all are true:
- inbox email
- not from the watched mailbox itself
- not a sent/outbound message
- appears to be from an individual human
- appears human-written, not newsletter/promo/bulk/no-reply
- low or medium risk
- no personal details
- no legal/financial/contract/compliance sensitivity
- no unusually delicate tone or negotiation risk
- not blocked by thread/follow-up spacing rules
- thread is not already in a waiting-on-them state

Business-opportunity signals include:
- consulting
- lead
- AWS
- Amazon
- data center
- colocation
- HPC
- AI cluster
- cooling
- power
- capacity
- migration
- design review
- due diligence

If business-opportunity signal is present and message is brief/vague, reply with a qualification-style response asking for:
- short description of the opportunity
- scope
- timeline
- client/company if shareable
- budget range and location if appropriate

Do not auto-reply if uncertain. In uncertain cases, draft mentally only and wait.
