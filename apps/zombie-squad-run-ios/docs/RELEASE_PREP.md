## Suggested App Store copy
### Subtitle
Zombie runner shooter with squad upgrades

### Promotional text
Build your squad, blast through zombies, and smash walls before they wipe out your team.

### Short description
A lane-based runner shooter where every shot matters. Recruit survivors, collect gun crates, and break walls before impact.

## Kid-safe / audience note
This current theme is zombie combat, so it fits casual action audiences better than young children. If you want a younger audience, reskin zombies into slime monsters, robots, or goofy mutants.
