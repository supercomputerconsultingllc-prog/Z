# Privacy Policy Draft

Zombie Squad Run does not require account creation to play the local browser version.

## Data stored locally
- best score
- coin balance

This data is stored locally in the browser on the user's device.

## Data collection
The current local version does not send gameplay data to a remote server.

## Future release note
If analytics, ads, cloud saves, or in-app purchases are added, this policy must be updated before release.
