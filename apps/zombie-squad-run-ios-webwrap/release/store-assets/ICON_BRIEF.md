# Icon Brief

## Goal
Create a bold icon that reads clearly at small size.

## Recommended concept
- dark background
- bright green zombie head or squad silhouette
- one strong accent color, yellow or orange
- no tiny text

## Avoid
- cluttered scene art
- too many characters
- full gameplay screenshot as icon
- thin outlines that disappear at small size

## Deliverables
- 1024x1024 master icon
- square safe composition
- optional alternate icon concepts A/B/C
