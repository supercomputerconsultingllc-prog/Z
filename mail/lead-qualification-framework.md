# Lead Qualification Framework

Use this for business-relevant data center consulting emails when a natural opportunity exists.

## Core qualification dimensions
- Project type
- Current environment
- Main challenge or constraint
- Timeline
- Budget range or budget signal
- Location / region
- Stakeholders / decision-makers
- Desired next step

## How to ask
Do not dump a long questionnaire. Ask only the smallest set of questions needed to move the conversation forward.

Preferred order:
1. What are they trying to accomplish?
2. What is the current environment or constraint?
3. What is the timeline?
4. If appropriate, what is the budget or scale signal?
5. Suggest the next step.

## Example next-step objectives
- clarify scope
- determine technical fit
- determine urgency
- determine whether a call makes sense
- determine whether the opportunity is real or speculative

## Example lightweight qualification reply
Hi [Name],

Thanks for reaching out. Yes, this is the kind of work we support.

If you send over a bit more detail on the current environment, the main challenge, and the timeline, I can help figure out the best next step.

Best,
Supercomputer Consulting
