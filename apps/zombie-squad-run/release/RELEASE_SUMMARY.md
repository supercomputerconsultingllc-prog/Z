# Zombie Squad Run Release Summary

## Current state
Zombie Squad Run is a playable browser prototype with:
- three-lane runner shooter gameplay
- zombie, wall, crate, and coin encounters
- squad growth
- firepower upgrades
- revive mechanic
- persistent best score and coin balance
