# Pricing Matrix

## Paid launch option
- $0.99: easiest impulse-buy entry
- $1.99: better revenue per download if visuals are polished enough

## Free launch option later
- rewarded revive
- ad-free unlock
- cosmetic squad pack
- cosmetic weapon pack

## Recommendation
Start paid if speed matters most.
Switch to free only if you are ready to support ad / IAP integration and update the privacy/store disclosures.
