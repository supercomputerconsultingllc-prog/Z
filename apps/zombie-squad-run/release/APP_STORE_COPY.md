## Full Description
Zombie Squad Run is a fast, pick-up-and-play action game where timing and target choice matter.

Run forward with your squad, fire into the lane you choose, and survive wave after wave of zombie chaos. Shoot zombie packs to grow your team, break walls before impact, collect gun crates to boost firepower, and use revives wisely when the run gets rough.

### Features
- Three-lane runner shooter action
- Squad growth through survivor rescues
- Gun upgrades and revive economy
- Coin-based progression loop
- Easy-to-learn controls
- Built for fast sessions and repeat runs
