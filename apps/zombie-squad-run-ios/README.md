# Zombie Squad Run iOS Scaffold

This is an iOS-ready SwiftUI scaffold for a mobile version of Zombie Squad Run.

## What is included
- SwiftUI app entry point
- game models
- view model with a simple game loop
- lane-based encounter board
- stats header
- control panel
- release-prep docs

## What still requires a Mac/Xcode environment
- create the Xcode project container
- attach these source files to the target
- set signing and bundle identifiers
- add assets, icons, splash screens, and StoreKit config
- archive and submit to App Store Connect

## Suggested folder mapping in Xcode
- `ZombieSquadRun/App`
- `ZombieSquadRun/Models`
- `ZombieSquadRun/ViewModels`
- `ZombieSquadRun/Views`

## Build target suggestion
- iOS 17+
- SwiftUI app lifecycle
